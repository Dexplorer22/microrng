# WheelV1 — Инструкция

## Запуск

**Через Docker (рекомендуется):**
```bash
docker compose up --build
```
Приложение будет доступно по адресу http://localhost:8000

**Без Docker:**
```bash
pip install -r requirements.txt
WHEELV1_ADMIN_USER=admin WHEELV1_ADMIN_PASSWORD=ваш_пароль WHEELV1_JWT_SECRET=секретный_ключ uvicorn main:app --host 0.0.0.0 --port 8000
```

> Если `WHEELV1_ADMIN_PASSWORD` не указан — авторизация отключена (небезопасно).

---

## Вход в админ-панель

Откройте http://localhost:8000 в браузере. Введите логин и пароль, которые задали при запуске.

---

## Создание колеса

1. Откройте вкладку **Колёса**
2. Внизу введите название в поле «Создать новое колесо»
3. Нажмите **Создать**
4. Выберите новое колесо в выпадающем списке вверху

## Настройка колеса

1. Выберите колесо в выпадающем списке вверху
2. Перейдите на вкладку **Настройки**
3. Заполните параметры:
   - **Цена спина** — сколько стоит один поворот колеса
   - **Целевая маржа** — желаемый процент прибыли (0.10 = 10%)
   - **Мин. маржа** — защита от убытков (0 = прибыль не уйдёт в минус)
4. В разделе **Тиры** можно настроить win-rate для разных уровней игроков
5. В разделе **Джекпот** — включить/отключить накопительный приз
6. Нажмите **Сохранить конфигурацию**

## Добавление призов

1. Перейдите на вкладку **Настройки**
2. В таблице призов нажмите **+ Приз**
3. Укажите название, стоимость и (необязательно) лимит выдачи
4. Нажмите **Сохранить призы**

## Запуск раунда

1. Перейдите на вкладку **Статистика**
2. Нажмите **Новый раунд**

---

## Управление пользователями

### Создать пользователя
1. Перейдите на вкладку **Пользователи**
2. Нажмите **+ Пользователь**
3. Введите логин, пароль и выберите роль:
   - **owner** — полный доступ
   - **readonly** — только просмотр
   - **editor** — редактирование колёс и призов, без управления пользователями
4. Нажмите **Создать**

### Удалить пользователя
1. Перейдите на вкладку **Пользователи**
2. Нажмите красную кнопку **Уд.** напротив нужного пользователя
3. Подтвердите удаление

## Роли

На вкладке **Роли** можно создавать свои роли с нужным набором прав. Доступные права:
- `wheels:read`, `wheels:write`, `wheels:delete` — управление колёсами
- `prizes:read`, `prizes:write` — управление призами
- `stats:read` — просмотр статистики
- `game:manage` — управление игрой (спины, раунды, сброс)
- `users:read`, `users:write` — управление пользователями
- `audit:read` — просмотр журнала действий
